# Learning Management System

A Laravel-based Learning Management System (LMS) for managing courses, students, and online learning activities.

## Features

- Course Management
- Student Enrollment
- User Authentication
- Admin Dashboard
- Teacher & Student Roles
- Assignment Management
- Online Learning System

## Tech Stack

- Laravel
- PHP
- MySQL
- Bootstrap
- Blade

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
